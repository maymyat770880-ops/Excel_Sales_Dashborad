# Interactive Sales and Profit Dashboard

## Project Overview
This project features an interactive Excel dashboard designed to analyze retail business performance. It visualizes key metrics like total sales, total profit, and customer distribution across different regions and categories.

## Key Features
* **Interactive Slicers:** Filter data by Year, Month, Category, and Product for dynamic insights.
* **Sales Analysis:** Monthly sales trends and sales distribution by state (USA).
* **Profit Tracking:** Annual profit comparisons and category-wise profit breakdowns.
* **Top 5 Customers:** Visualization of the highest-contributing clients.

## Tools Used
* **Microsoft Excel:** Pivot Tables, Slicers, and Advanced Charting.
